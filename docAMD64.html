Installing LNP OS on AMD64

LNP OS for AMD64 is a high-performance, non-Linux based operating system designed for modern desktop and workstation environments. It uses our custom kernel that delivers efficiency, security, and near-native hardware performance.

Step 1: Download the ISO

Download the LNP OS AMD64 ISO from our official download page.

Step 2: Create Bootable Media

Use a reliable tool like Rufus or Balena Etcher to flash the ISO to a USB drive.

Recommended: Use a USB 3.0 device for faster installation.

Example (Rufus Settings)

  • Partition scheme: GPT
  • Target system: UEFI (non-CSM)
  • File system: FAT32

Step 3: Boot into Installer

Plug the USB into your AMD64 machine and reboot. Enter your BIOS/UEFI settings and set the USB as your first boot device.

On most PCs, press F12, F10, or DEL during startup to access boot options.

Step 4: Installation

Once the installer loads, choose between a clean install or dual booting with Windows. If you choose dual boot:

  • Ensure Windows is installed on NTFS partition
  • Use the installer’s guided partitioning tool
  • Click "Install LNP OS Alongside Windows"

Step 5: Post-Install Setup

After rebooting, LNP OS will automatically detect your hardware and configure necessary drivers. No manual driver setup is required.

If you experience boot issues, access your UEFI boot manager and select LNP OS Bootloader.

Boot Loader Note

LNP OS uses a custom bootloader, not GRUB. If you're manually adjusting partitions, avoid modifying the LNP_BOOT partition.

Minimum Requirements

  • Processor: AMD or Intel 64-bit CPU
  • RAM: 4GB minimum (8GB recommended)
  • Storage: 25GB free disk space
  • UEFI Firmware
