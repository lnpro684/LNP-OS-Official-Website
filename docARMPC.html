LNP OS Installation Guide for ARM64 PC

LNP OS for ARM64 PC brings optimized performance for ARM-based desktop and server devices. Designed to be lightweight yet powerful, it supports most ARM SBCs (Single-Board Computers), laptops, and servers.

Step 1: Download the ARM64 PC ISO

Visit our official download page to get the latest LNP OS ARM64 PC ISO.

Step 2: Create a Bootable Media

Use Balena Etcher to flash the ISO to a microSD card or USB drive.

Step 3: Boot Your ARM PC

Insert the bootable media and power on your ARM64 device. Some devices require pressing a key (e.g., ESC or F2) to access the boot menu.

Step 4: Install LNP OS

Follow the guided setup wizard. You may choose to:

  • Install on a new partition
  • Install alongside existing OS
  • Erase and install fresh

Minimum System Requirements

  • 64-bit ARM CPU
  • 2GB RAM minimum
  • 16GB of free storage
  • UEFI/firmware boot supported

Dont worry about drivers, LNP OS is searching it automatically.

Enjoy a stable and modern environment with native support for most ARM peripherals.
