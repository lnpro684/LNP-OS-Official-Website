Troubleshooting LNP OS on ARM64 PC

Issue 1: System Won’t Boot

Check your board's documentation for boot key and verify bootable media is properly flashed. 

lsblk

Issue 2: Display Not Working

Try another HDMI port or resolution setting. Some boards (like Raspberry Pi) require config updates. 

sudo nano /boot/config.txt

Issue 3: USB Devices Not Recognized

Make sure you’re using a powered hub or try switching USB ports. Also, run: 

dmesg | grep usb

Issue 4: Cannot Access Wi-Fi

Use: 

sudo lnp-network --repair

For some boards, Wi-Fi firmware must be manually added. Refer to your device’s compatibility notes.
