Troubleshooting LNP OS on AMD64

Issue 1: Black Screen After Installation

Check if secure boot is disabled in UEFI settings. Also verify boot priority is set correctly.

Issue 2: Dual Boot Not Appearing

Use UEFI boot menu (F12, ESC, or DEL) and check if LNP OS Bootloader exists.

Issue 3: No Network or Audio

Go to Settings > Device Manager. Run: 

sudo lnp-drivers --scan
